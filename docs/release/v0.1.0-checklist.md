# v0.1.0 Release Candidate Checklist

Status values:

- `[x]` verified in this workspace.
- `[ ]` not verified.
- `[~]` partially implemented or locally verified with limitations.

## Build And Packaging

- [x] App builds for Apple Silicon.
- [x] Release archive is generated as `Portnado-v0.1.0-darwin-arm64.zip`.
- [x] Release archive SHA-256 is generated.
- [x] Release archive structure and checksum are verified by
  `scripts/verify-release-artifact.sh`.
- [x] Archive contains one `Portnado.app`.
- [x] App embeds `portnado` and `portnado-daemon`.
- [x] Embedded CLI reports `portnado 0.1.0`.
- [ ] App is Developer ID signed and notarized.
- [ ] Release has been published with explicit approval.
